This map depicts fish passage barriers that have been corrected on streams known to have coho, along with the known distribution of coho salmon in Washington.

The coho distribution file is from streamsnet.org (January 2019). This file was edited to decrease the size by removing unecessary attributes and by using mapshaper.org to simplify it.

The corrected fish barriers passages is from the WSDOT and was retrieved from: https://geo.wa.gov/datasets. Only barriers with the species attribute containing coho were selected.

Both layers were reprojected into GCS_WGS_1984 and converted into GeoJSON using QGIS
